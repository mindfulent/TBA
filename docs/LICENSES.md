## Mods Requiring Attention

### All Rights Reserved (Check Modpack Permissions)
These mods explicitly allow modpack inclusion:
- **Excessive Building** - "Feel free to include in modpacks"
- **Universal Sawmill** - "You can use this in your modpacks"
- **Simple Voice Chat** - Generally allows modpack distribution
- **Flan** - Widely used in modpacks
- **JourneyMap** - Check journeymap.info for specific rules
- **Chipped** - Custom Terrarium license

### Custom Licenses
- **Handcrafted**, **Athena CTM** - Terrarium License
- **[Let's Do] series** - Custom license by satisfyu
- **Moonlight Lib** - LGPL with dependency clause

### Sodium's Polyform Shield
Sodium uses Polyform Shield 1.0.0 which prohibits competitive use. This is fine for:
- Private servers
- Non-commercial modpacks
- Educational use

NOT allowed for: Creating a competing optimization mod/product.

## Compliance Notes for Multiplayer Server Use

1. **Non-commercial server**: All licenses are compatible with non-commercial/private server use.

2. **Attribution**: Consider crediting mod authors in server documentation or on website.

3. **Distribution**: When distributing the .mrpack file:
   - Prefer linking to official download sources (CurseForge, Modrinth)
   - Most mods allow modpack redistribution
   - ARR mods generally allow modpack inclusion but not standalone redistribution

4. **Source availability**: For copyleft licenses (GPL, LGPL), source code is available on the respective GitHub/GitLab repositories.

5. **Commercial use**: If the server becomes monetized (donations, subscriptions):
   - Review any "All Rights Reserved" mods' specific terms
   - v0.9.21 removed Jade (CC BY-NC-SA 4.0) in favor of WTHIT (MIT) for this reason
